---
title: Connect your RAG context 
description: To integrate RAG context into Maxim, you need to create a context source and add your RAG context API endpoint. This context source can then be utilized in prompts and workflows for inferencing, enabling the model to access and use the relevant context during processing.
---
## Create a context source
- Navigate to "Context Sources" on the left navigation bar.
- Click on the "+" icon to create a context source.

<img src="./Library/context-source/context-source-connect/img1.png" alt="http_workflow" />

- In the dialog box that appears, provide a name for the context source.
- Add RAG context endpoint
- Add necessary headers and parameters to your request and save the context source
<img src="./Library/context-source/context-source-connect/img2.png" alt="http_workflow" />
