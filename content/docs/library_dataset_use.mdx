---
title: Using a dataset for evaluation 
description: Dataset can be used for evaluation of a prompt or a workflow 
---
Upon clicking the test buttons in either of these flows a right side panel of test config menu appears, where you can select your dataset to be used for testing.

<LoomVideo url="https://drive.google.com/file/d/1uUjmDu-2IlOT_HZfXmnyzs-WK4LLcCqz/preview" />


As demonstrated in the video, in the final report the input from the dataset serves as a query to the model. If a context source has been added, it appears as a column alongside. Depending on the evaluators chosen we evaluate the generated output or the generated output with respect to expected output (ground truth) and we also have evaluators judging the quality of the retrieved context if context source is attached.
